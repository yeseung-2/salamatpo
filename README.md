# SalamatPo

SalamatPo는 필리핀 만성질환자와 보호자가 약 이름, 병명, 지역, 자격 정보를 입력하면 무료약, 정부지원, 저가 약국, 재고 가능 경로를 추천받을 수 있도록 돕는 약 접근성 플랫폼입니다.

## Core Features

- 약 이름, 병명, 처방전 기반 약 찾기
- GAMOT, RHU, Senior Discount, PWD Discount 등 정부지원 가능성 확인
- 예상 비용, 준비물, 거리, 재고 가능성 기반 추천 결과 제공
- 지역 기반 커뮤니티를 통한 재고 및 대기시간 정보 공유
- 복약 시간 설정 및 재구매 시기 관리

## Tech Stack

### Frontend

- Next.js
- TypeScript
- Tailwind CSS
- Axios

### Backend

- FastAPI
- Python 3.11
- PostgreSQL 예정
- SQLAlchemy 예정

### Deployment

- Frontend: Vercel 예정
- Backend: Railway 예정
- Database: Supabase Postgres 예정
